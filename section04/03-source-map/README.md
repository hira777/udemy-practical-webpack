# ソースマップを生成してデバッグに役立てる

「ソースマップを生成してデバッグに役立てる」を終了時点のサンプルコード。

## 動作環境

- Node.js: v12.18.1
- npm: v6.14.5

## セットアップ

このディレクトリ上で以下のコマンドを実行してください。

```shell
npm ci
```

## 使い方

以下のコマンドを実行すれば、開発用の設定で webpack が実行され、バンドルされたファイルが出力されます。

```shell
npm run dev
```

以下のコマンドを実行すれば、本番用の設定で webpack が実行され、バンドルされたファイルが出力されます。

```shell
npm run build
```
